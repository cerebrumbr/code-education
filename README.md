Principais instruções para este commit
======================================

Os arquivos aquí adicionados fazem parte do curso do portal code-education

01° Estágio
-----------

Criei o repositório no github com o nome code-education

02° Estágio
-----------

Criei uma pasta localmente, e em seguida adicionei o arquivo README.md.
Após isso iniciei o git digitando:
git init

Para verificar o status digitei:
git status

Adicionei o README.md com o comando:
git add .

Criei uma pasta com o nome 01-Git, com um arquivo Trabalhando com github.txt dentro dela
adicionei a pasta e o arquivo com o comando:
git add .

comitei com o comando:
git commit -m "Primeiro commit, projeto pratico - code.education"

03º Estágio
-----------

Enviei para o repositório code-education com os códigos:
git remote add origin [https://github.com/candidosouza/code-education.git](https://github.com/candidosouza/code-education.git)

E depois com:
git push -u origin master
